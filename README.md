2. Prompt Type: Comparative Analysis Prompt

Prompt:

“Compare how a rule-based automation system and a machine-learning-based predictive maintenance system would respond when monitoring equipment performance in a manufacturing plant.”

Expected Output:

“A rule-based automation system would respond based on predefined conditions. For example: ‘The machine temperature has exceeded the preset limit. The system will shut down the equipment and send an alert to the maintenance team.’ This approach is simple and reliable but reacts only after a rule is triggered.

A machine-learning-based predictive maintenance system, however, analyzes historical and real-time sensor data from IoT devices. It may respond like: ‘The vibration and temperature patterns of Machine A indicate a possible failure within the next 48 hours. Maintenance is recommended to prevent downtime.’ This approach is more advanced because it predicts issues before they occur and helps reduce unexpected production interruptions.”

Scenario Prompt:

“A factory manager notices that one of the production machines is running slower than usual and asks the system:
‘Can you check the performance of this machine and tell me if there might be a problem?’”
